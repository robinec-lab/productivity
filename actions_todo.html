Génère un fichier CSV avec toutes vos actions. Compatible Google Sheets, Excel, LibreOffice.


    

      Importer dans Google Sheets :

      1. Téléchargez le CSV ci-dessus

      2. Ouvrez sheets.google.com → créer un nouveau tableau

      3. Fichier → Importer → Choisir le fichier

      4. Séparateur : Virgule → Importer les données
